# Gliding Platformer

Platformer where you glide with mechinics inspired by
the level "An Artificer's Accent" in the Celeste 2024 Secret Santa Collab and
the elytra in Minecraft

## Controls

* Move:arrow keys or wasd

* Glide: Shift

* Jump: Space or Z

* Respawn: R

Tileset: [Kenney New Platformer Pack](https://kenney.nl/assets/new-platformer-pack)

Made with the Godot Engine

Made for Hack Club Siege Week 13 with the theme of "Winter"

## To play:

Run the web version or download the desktop buils from the game's page on [itch.io](https://xanderath.itch.io/gliding-platformer)

or

1. Download Godot from https://godotengine.org/

2. Download the repo using
``git clone https://github.com/Emarald64/gliding-game.git``

4. Import the folder you cloned the repo into Godot
